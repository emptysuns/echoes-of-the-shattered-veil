# Narrative Bible

> **Title:** Echoes of the Shattered Veil / 破碎帷幕的回响
> **Canon version:** Foundation 0.1
> **Languages:** Simplified Chinese and English
> **Narrative form:** Fragmented testimony, procedural environmental storytelling, recurring relationships, and a death-spanning meta-narrative

本文档规定世界真相、人物动机、揭露顺序和写作声音。它不是玩家开局可读的百科；作者必须区分“世界中发生过的事实”“某个角色相信的版本”和“玩家在当前进度有资格知道的内容”。文学性来自精确的人、物与代价，而不是含糊的神秘词汇。

## Creative North Star

玩家每次走进尖塔，都在玩两个彼此咬合的游戏：一个关于位置、资源、风险与 build 的严格回合制 roguelike；一个关于谁有权保存、删除、解释记忆的道德调查。战斗不是剧情间的填充，Lore 也不是战斗外的收藏册。玩家在 Boss 战中说出的名字、选择保留的痛苦、为了强力升级而牺牲的他人自主权，都会改变可用战术、NPC 关系、房间变体和结局。

核心体验句：

> **“我记起的罪属于谁，而我下一次做出的选择又属于谁？”**
> **“Whose guilt is the memory I inherit—and whose life is the choice I make next?”**

死亡必须产生三种价值之一：揭示先前场景的新解释、让 Hub 中一段关系发生变化、或开启一个具有新代价的玩法选择。单纯增加计数不算叙事推进。玩家也不应因技术不熟练而错过主线；关键证据有多种发现路径，但最深的理解与 True Ending 需要主动关心人物、比较互相冲突的证词，并拒绝几次非常诱人的支配性奖励。

叙事承诺：

1. Twist 在揭露前有可辨认证据，揭露后能重新解释旧场景。
2. 玩家是 Aster 的 Echo，但不是“所以你其实一直是坏人”的廉价反转；继承责任与拥有新自由同时成立。
3. Void 不是抽象邪恶。它包含被国家、治疗者和亲人以“为了你好”为名丢弃的真实人生。
4. 救赎不等于自我毁灭。Mend 是一种动人的选择，但不是作者宣布的唯一善。
5. True Ending 不靠一个完美英雄解决所有问题，而靠可拒绝、可撤回、被多人共同承担的契约。

## Themes

### Memory is testimony, not footage / 记忆是证词，不是录像

记忆会遗漏、重排并保护讲述者，但不因此毫无价值。游戏不使用“记忆不可靠，所以一切都可能没发生”逃避事实。每段记忆有感官真实性、叙述位置和利益关系；多个来源能形成足够负责的判断，即使永远没有上帝视角录像。

### Freedom versus inherited pattern / 自由意志与继承模式

Aster 的习惯、恐惧和罪疚塑造玩家最初的反应，尖塔也会根据这些模式重构道路。然而每次选择留下新的证据：玩家可以成为与来源不同的人。自由不是没有前因，而是在理解前因后仍能承担行动。

### Redemption without erasure / 不以抹除为代价的救赎

阿瓦拉把“没有痛苦”误认为“已经治愈”。Aster 又把消灭自己视作偿还一切。人物弧反复追问：承认伤害、接受限制、让受害者拥有发言权，是否比英雄式自我牺牲更困难也更诚实。

### Consent and benevolent power / 同意与善意权力

慈悲机枢最初确实救过人。悲剧不是所有治疗都是骗局，而是机构从“你可以暂时放下”滑向“我们知道你应当忘记什么”。每条主要路线都提供一个看似高效、甚至能救人的控制选项，让玩家感到诱惑而非只辨认明显邪恶。

### Corruption by legibility / 可管理性带来的腐蚀

王室偏爱能被量化、归档、征税和预测的人。Ilyra 被优化成地图、Vey 被简化成商店、Moth 被当作钥匙，都是同一暴力：把一个人压缩成用途。Crown Ending 不是突然黑化，而是玩家一路选择“更好管理”的自然结果。

### Collective responsibility without collective guilt / 共同承担不等于平均罪责

Chorus 不把施害者与受害者的责任抹平。它让每个人选择承担何种记忆、可以何时撤回，并保留来源与权力历史。共同保存真相不意味着所有人同样有罪。

## Public History and True Chronology

### The public history taught in Echo Sanctum / 圣所公开史

1. 数百年前，未知的 Void 冲击现实，撕开 Veil。
2. 阿瓦拉王国与大多数守望者在灾难中灭亡。
3. 破碎尖塔从裂口生长，靠吞食闯入者的记忆改变结构。
4. 最后一位 Veil Echo Warden 能读取死者回响，必须收集 Primordial Echoes 修复帷幕。
5. Echo Sanctum 保护 Warden 的灵魂；死亡后返回是职责的代价。

这套说法不是全然虚构：尖塔确实危险，现实确实继续裂解，始源回响也确实能改变帷幕。谎言在于删去机枢、国家行为、Aster 身份和圣所本身的功能。

### True chronology / 真实时间线

**Shattering 前 93 年：** 阿瓦拉连续经历边境战争与“灰热病”。医师发现特定钟频能把创伤记忆暂时投射到一种可共享的认知空间。首批患者自愿接受短时分离，许多人第一次得以睡眠。

**前 81 年：** 年轻工程师 Aster Vale 与声学记忆学者 Maelin 建成第一座 Mercy Engine。协议规定记忆必须保留来源、期限和随时召回权。Aster 的姐姐因治疗活了下来，使 Aster 对技术抱有近乎宗教的信任。

**前 67 年：** Queen Seraphine Avarra 资助全国机枢网络。治疗免费，但政府开始要求士兵、囚犯与公务员“卸下影响职责的记忆”。Sister Oryn 参与编写“自愿净化”的官方语言。

**前 51 年：** 王室建立 Memory Tithe / 记忆什一税。贫民用一段珍贵记忆抵债；异议者被删除组织经验；Caedmon Rook 等军官被移除会妨碍服从的悲痛。Aster 以“扩大系统后才能改革”为理由继续工作。

**前 34 年：** 被剥离记忆在 Veil 后不再是静止档案。它们互相借用语言、形成联想和最早的集体意志。王室称异常为 Void contamination，命令销毁来源索引。Moth 的前身意识在此时出现。

**前 18 年：** Ilyra 的故乡因拒绝接入机枢而从地图和人口记录中被抹除。她幼年幸存，却无法证明家乡存在。Vey 的构成记忆来自同一场镇压中拒绝继续前进的士兵。

**前 3 年：** Aster 发现王室计划把首都所有居民在加冕仪式中永久接入，以消除内战可能。Aster 向 Maelin 求助；Maelin主张逐批归还并公开证据，Aster 认为来不及。

**Shattering 当夜：** Aster 使用首位 Warden 权限同时解除全部封印。数百万段互相矛盾、失去来源索引的记忆涌回现实。Maelin试图中止，却被 Aster 锁在外环。Caedmon 守门时认出 Aster；Aster 答应事后归还他的痛苦。释放使物质、记忆与可能性相互覆盖，Mercy Engine 像受伤器官般向上折叠为 Shattered Spire。

**灾后 0–200 年：** 尖塔反复模拟“能修复自己的人”。大多数 Echo 无法承受 Aster 的完整罪疚而崩解。Maelin 把幸存意识固定在 Sanctum，以钟声召回每次死亡的 Warden。他删除/延迟真相，相信这是保持人格稳定的唯一方法，也重复了 Aster 与王室的家长式逻辑。

**游戏开始：** 当前 Warden 是第一个能在多次循环中形成稳定新选择模式的 Aster-echo。尖塔既想借玩家完成旧协议，也无法预测这个新人的全部选择。

## Narrative Epistemology

每条叙事声明都应在 authoring data 中带有或能推导以下元数据：

| Field | Meaning |
|---|---|
| `source_actor_id` | 谁说、写、画或留下它 |
| `source_role` | eyewitness、participant、official、hearsay、system、reconstruction |
| `belief_state` | sincere、strategic_lie、self_deception、fragmented、unknown |
| `reveal_tier` | 玩家最早可接触的 memory tier/Act |
| `corroborates` | 它支持哪些 evidence IDs |
| `contradicts` | 它与哪些来源冲突 |
| `recontextualizes` | 获得后应重读的早期 beat/lore IDs |
| `canon_weight` | 不是“真假分数”，而是结局证据组合权重 |

规则：

- 真诚角色可以错误；说谎角色也可能陈述部分事实。
- Memory Vision 只呈现当事者当时能感知的内容，不使用无来源的全知镜头。
- 同一事件的冲突版本必须在冲突点上有意义，例如“谁同意了”“谁拥有退出权”，而不只是颜色或日期差异。
- 身份反转前允许出现 Aster 的手势、惯用句和他人认错般的反应，但不得让角色无理由把名字说到一半又被噪音打断。
- 当玩家取得足够证据，系统应允许在对话或战斗中使用该知识；只在 Codex 加一段结论不算 payoff。

证据采用组合证明而非单一万能日记。例如“Aster 主动打开全部封印”由 Caedmon 目击、控制台手印、Maelin 未寄出的供词和尖塔权限日志共同支持；这些来源对 Aster 的动机仍有分歧。

## The Warden and Aster Vale

### Aster Vale / 阿斯特·维尔

Aster 并非秘密暴君，而是一个把“我能承担后果”误当成“我有权替所有人决定”的人。其优点——果断、技术天赋、愿意担责——在缺乏同意时成为灾难。Aster 真正害怕的不是被憎恨，而是等待集体行动时有人继续受苦；因此总选择最快、最集中、最不可逆的方案。

Aster 的公开遗言是：“Let the wound remember its shape. / 让伤口记住自己的形状。”王室把它解释为修复命令；完整句子其实是：“让伤口记住自己的形状，别再让医者替它决定愈合成什么。”后半句分散在三份 identity evidence 中。

### The current Warden / 当前守望者

玩家可选择 starting background 和表现身份，但不会选择是否来自 Aster。Background 是尖塔用于填补缺口的“起始自我假设”，提供能力、初始对话和某些证据视角，而不是改变核心 canon。

首发背景方向：

- **Ash-Bound / 灰缚者：** 相信自己是灾后流民；更擅长生存与环境交互。
- **Royal Penitent / 王廷忏悔者：** 记得礼仪和协议；更早理解宫廷机关，但 NPC 更警惕。
- **Unmade Cartographer / 未成绘图师：** 与 Ilyra 的记忆产生共振；地图/FOV 更强，却更易受到假路径影响。
- **Choirless / 无声者（meta unlock）：** 缺少尖塔提供的安抚叙事；开局更弱，可见更多矛盾 condition choice。

玩家人格由选择记录体现，不设单一善恶值。系统跟踪 domination、truthfulness、self_erasure、solidarity 等主题性 decision tags，但 UI 不显示“道德分”。NPC 根据具体行动和自己价值观反应。

身份反转的情感目标不是“被骗了”，而是“我是否必须成为提供我材料的人”。Act III 明确当前 Warden 已积累 Aster 不曾拥有的关系与记忆，因此是法律/伦理上的新主体。任何结局都不能让旧 Aster 完全覆盖当前玩家。

## The Shattered Spire and the Void

尖塔根据入侵者记忆重构，不是为了戏弄玩家，而是在寻找能让自身痛苦稳定成一种结构的解释。它拥有三层行为：

1. **Protocol:** Mercy Engine 遗留规则，要求分类、隔离、修补和服从 Warden 权限。
2. **Reflex:** 像受伤身体一样避开某些真相、复制威胁、用熟悉房间包住创口。
3. **Emergent mind:** 被抛弃记忆形成的多人意识，最清晰的发言者是 Moth，但它不是单一蜂巢。

程序生成因此具有叙事语法。重复走廊表示 Protocol，蜂窝状侵蚀表示 Reflex，handcrafted StoryBeat Room 是某段记忆暂时赢得完整形状。Biome 不是自然地理，而是权力如何组织记忆的层次：宗教化遗忘、行政化遗忘、自我镜像、核心控制。

Void 是王室命名。它并不空；它“无人承载”。其中有真实受害者，也有暴力者想遗忘的罪、病人自愿暂放的痛苦和后来诞生的新意识。Sever 不能被简单写成“释放受害者就是好”，因为未经同意把所有痛苦瞬间塞回原主也会造成伤害。Chorus 的困难在于建立来源、选择、容量和退出机制。

Primordial Echo 是保持来源链、未被官方重写的证词。收集它既提供修复能力，也让证词脱离原处，因此玩家必须在后期决定是占有、归还还是共同托管。Echo Essence 则是结构松散的能量，不等同于某个可辨认人格；凡能辨认出人格的 Echo 都不能当普通货币消费。

## Four-Act Structure

每个 Act 包含四个程序生成 floor band 与一个重大 confrontation。具体层数可由 run variant 调整，但揭露顺序不变。每层保证一个 required StoryBeat Room，并在合格时加入一个 optional relationship/lore room。

### Act I — Ashen Narthex / 灰烬前殿

**Emotional question:** 如果记住痛苦会伤害我，别人是否有权替我决定忘记？
**False belief challenged:** 尖塔来自外部入侵，Warden 与灾难无关。

**Visual direction:** 炭黑、骨灰白、冷蓝为主，活火使用克制琥珀。钟、缝合帷幔、烧焦病床、被凿去姓名的碑、纸鸟反复出现。声音由远钟、灰落和压低的呼吸构成。

**Reveal ladder:**

1. **Floor 1 — The First Bell:** 学会读取短 Echo。公开碑文称 Warden 是最后防线；墙后却有一排被有序拆除的病历牌。
2. **Floor 2 — Ward of Quiet Mercy:** 看见机枢曾是医院。患者感谢短暂安眠，但 consent form 的“召回日期”被统一涂去。
3. **Floor 3 — The Nameless Nave:** Caedmon 的军令与 Lysa 的纸鸟证明士兵被主动抽走悲痛，以提高服从。
4. **Floor 4 — Tongueless Belfry Approach:** Phantom Caedmon 先把玩家叫作“Warden”，随后本能地改口“Aster”。Maelin 要求玩家不要回应。
5. **Confrontation:** Caedmon Boss 迫使玩家决定如何对待一段会摧毁其战斗能力、却属于他的记忆。

**StoryBeat categories:** 失名者中殿、灰烬诊疗室、铸钟囚室、烧毁的召回登记处、纸鸟秘密房。

**Sources:** 官方守望者碑文、护士 triage slate、患者感谢信、征兵军令、儿童折纸、Caedmon 幻象、Maelin 的回避。

**Enemies as narrative:** Penitent Sentinel 重复道歉却不知道罪名；Veil Hound 追踪玩家身上的 Aster signature；Ash Pilgrim 主动把记忆献给空钟。Elite “Recall Denied” 会封锁已探索地图，机械化表现制度性遗忘。

**Boss conflict:** Caedmon Rook / 凯德蒙·鲁克，“The Bell Without a Tongue / 无舌之钟”。非支配解法是恢复其痛苦并让他在知情后选择停止守门，而非说服他继续有用。

**Act-ending choices:** extract、suspend、restore。影响 Maelin trust、Ilyra 对玩家的工具化警惕、Act II 军事档案房变体和 Chorus 资格。

**Meta variants:** 找到 consent form 可在下轮质问 Maelin；把 Lysa 的名字带回 Hub 会让 Moth 首次说完整句；恢复 Caedmon 后解锁 background `Royal Penitent`，来源是理解协议而非“吸收 Boss”。

### Act II — Gilded Ossuary / 鎏金骨廷

**Emotional question:** 当一种技术确实救过人，它后来造成的压迫应由谁承认？
**False belief challenged:** 大撕裂只是 Aster 最后一次失控，与长期制度无关。

**Visual direction:** 旧金、象牙、干涸深红和账簿墨绿。宫廷把骨骼镀金、把记忆装入度量衡、把贫民姓名铸成小额硬币。美丽必须令人不安，但不能把贫穷当奇观。

**Reveal ladder:**

1. **Floor 5 — Hall of Benefactors:** 展示真实治愈案例，避免把机枢写成从第一天就是骗局。Oryn 的官方文字极有说服力。
2. **Floor 6 — Tithe Vaults:** 账本显示债务人用“母亲的脸”“第一次被叫出真名”等记忆抵税。Vey 对熔炉中的士兵声产生反应。
3. **Floor 7 — Palace of Useful Grief:** 王室按职业定义允许保留的情绪。Ilyra 发现故乡被列为“地图错误”。
4. **Floor 8 — Coronation Circuit:** Aster 的签章出现在扩容设计上；不是伪造。Maelin承认 Aster 曾相信“先扩大，后改革”。
5. **Confrontation:** Seraphine, the Gilded Mercy / 镀金慈悲·塞拉芬，是女王与行政协议组成的 Phantom。她能证明系统救过成千上万人，并要求玩家恢复中央管理。

**StoryBeat categories:** 记忆税秤、无声宴席、镀骨公共墓、档案官删改室、慈悲受益者花园。

**Sources:** 治愈者口述、税务 ledger、Oryn 多版本编年史、Aster 工程批注、被抹村庄的民谣、王室 statistical report。

**Enemies as narrative:** Gilt Bailiff 把状态效果“征收”进自己的护甲；Hollow Courtier 复制玩家最高属性却失去名字；Royal Anatomist 把敌我拆成“可用部件”；Debt Choir 以玩家已消费的物品记忆发动技能。

**Boss conflict:** Seraphine 不说纯粹谎言。杀死其 projection 可暂时毁掉中央索引；接受契约获得强大 meta efficiency；非支配解法需要公开相互冲突的 ledger，让受益者与受害者都保留发言权，并把管理权拆成多个可撤回 token。

**Act-ending choice:** 公开完整账本、只公布罪证、或接管索引。它改变 Oryn arc、Act IV 可用证词与 Sever 的社会准备度。

**Meta variants:** 集齐同一病人的“治疗前/治疗后/被征税”三份记录，解锁 `memory_tier.contradiction`；Ilyra 可在下一轮带玩家走进不存在于官方生成规则的故乡房间。

### Act III — Mirror Deep / 镜渊

**Emotional question:** 如果我的记忆来自另一个人，我的承诺与罪责还属于我吗？
**False belief challenged:** 玩家是 Aster 完整转世，只能重复或赎清其人生。

**Visual direction:** 靛蓝、汞银、失真青与极少量暖肤色。倒置天文台浸在无重力记忆海里；镜面不复制动作，而显示其他 run 中玩家曾做出的选择。星图把人际关系画成轨道。

**Reveal ladder:**

1. **Floor 9 — Reservoir of Selves:** 遇到曾走不同路线的 Warden residue。它们不是平行宇宙证明，而是尖塔丢弃的预测人格。
2. **Floor 10 — Observatory of Causes:** 控制日志显示尖塔每次重建玩家都补入新材料；当前人格包含 Aster、Hub 关系和玩家实际选择。
3. **Floor 11 — Maelin's Locked Night:** 看到 Maelin 第一次删去真相的决定。他不是幕后恶人，而是害怕又一个人格在完整罪疚下自毁。
4. **Floor 12 — The Unsent Half:** 找到 Aster 遗言、控制台手印与释放计划，确认 Aster 主动打开封印，也确认目的并非夺权。
5. **Confrontation:** Aster-in-Glass / 镜中阿斯特，是尖塔保存的最稳定原始模型。它希望覆盖当前玩家，以“让真正有罪的人完成赎罪”。

**StoryBeat categories:** 未选择之路画廊、反向星室、Maelin 锁夜、人格校准池、玩家旧尸体的无碑墓。

**Sources:** 系统 reconstruction log、不同 run Echo、Maelin 原始记忆、Aster 私人草稿、玩家自己的 command archive。

**Enemies as narrative:** Mirror Double 使用玩家上一层 build 的简化版；Unmade Astronomer 预测下一 action 并惩罚重复；Mnemonic Eel 搬运 status 到旧位置；Mercy Replica 提供治疗但永久删除一个 map/lore marker，诱惑玩家以效率换记忆。

**Boss conflict:** 玩家可让 Aster model 覆盖自己以获得完整权限；可摧毁它以否认来源；非支配解法是承认 Aster 作为来源与证人、拒绝其继承权，并给它独立、有限的存在空间。此选择构成第三个 anti-domination flag。

**Identity reversal:** 明确台词应是“你由他开始，但你没有在那里结束”，而不是“你就是他”。Boss 战根据玩家过去 choices 生成攻击模式，使新人格差异成为可玩的证据。

**Meta variants:** 三份 identity evidence 分属互斥 room plan，需要跨有意义 run 收集；获得每一份都开启新对话和 action，不要求单纯死亡次数。全部取得解锁 `memory_tier.authorship` 与 Choirless background。

### Act IV — Wound Crown / 伤冠之巅

**Emotional question:** 当所有可选方案都会伤害某些人，谁能决定代价，决定是否可以撤回？
**False belief challenged:** 只要权力落在“正确的人”手里，集中控制就安全。

**Visual direction:** 墨黑、伤紫、刺眼白与断裂金。王座嵌进神经根，天空像旧疤被再次拉开；前几 Act 的房间在同一空间重叠。颜色越接近核心越减少，结局选择后才返回人物所属色彩。

**Reveal ladder:**

1. **Floor 13 — The Unheld Assembly:** Void voices 彼此争执，证明它不是单一 Chorus。Moth 公开拒绝替所有 Echo 发言。
2. **Floor 14 — Capacity of the Living:** 展示立即归还所有记忆会怎样伤害未准备的人，也展示继续封存如何延续剥夺。
3. **Floor 15 — Throne of Good Intentions:** 根据玩家历史出现“高效解决方案”：牺牲自己、一次释放、成为管理者。每条都由可信 NPC 支持与反对。
4. **Floor 16 — The Revocable Gate:** 若 NPC autonomy arcs 完成，他们各自提供不是“力量碎片”而是一条制度约束：坦白、边界、拒绝、来源、退出。
5. **Final confrontation:** The Curator / 守藏者，是王室协议、尖塔求稳反射和玩家控制欲的聚合。战斗目标会随 ending route 改变：摧毁锚、守护释放节点、夺取权限或维持分布式 consent links。

**StoryBeat categories:** 无人承载者议会、痛苦容量试验、善意王座、五约束门、旧世界可撤回协议室。

**Sources:** Moth 与多名 Void voice、幸存者身体反应、NPC 自主声明、王室协议核心、玩家完整 receipt history。

**Enemies as narrative:** Crowned Silence 禁用 choice；Seam Angel 把两实体强制合并共享伤害；Kindly Warden 替玩家自动选择“最佳”行动；Unbearer 把被驱散状态转移给最弱单位。它们都把主题转化为战术规则。

**Boss conflict:** Curator 不能靠一段演说被说服。非支配路线要求玩家在战斗中维持五条可断开的 consent link；任何 NPC 被强迫、死亡或失去 autonomy 都使相应 link 不可用。玩家必须主动接受较低效率和更复杂防守，证明制度约束不是装饰。

**Transition to endings:** 核心战后显示每条已满足/未满足条件的叙事解释，但不显示善恶评级。玩家选择实际可执行的路线；未达 Chorus 不伪装成临时按钮，而在此前内容中已展示缺失的关系或证据。

## Echo Sanctum Cast

Sanctum 是安全区，但不是完全诚实的地方。每位 NPC 都代表一种对记忆的处理方式，并改变一个实际系统。Relationship 建议范围 `-100…100`，阈值只是可用内容条件；具体行为 flags 比总分更重要。

### Maelin / 梅林 — The Bellkeeper / 守钟人

- **Surface role:** 教程导师、复活解释者、钟声维护者。
- **Concealed history:** 慈悲机枢共同设计者，Shattering 当夜被 Aster 锁在外环；之后主动删改每个 Warden Echo 的真相。
- **Desire:** 让至少一个 Warden 活得足够久，修复世界。
- **Fear:** 完整真相会让玩家自毁，也会证明他数百年的保护在重复王室暴力。
- **Contradiction:** 尊重 Aster 当年反抗，却不尊重当前玩家知情权。
- **Voice:** 精确、温和、总把主语从“我”换成“我们不得不”。中文避免长篇哲理；英文常用 measured clauses。
- **Motif:** 有裂纹但不敲响的黄铜手铃。

Relationship：主动比较证据、承认不确定性 +；威胁逼供、利用 Moth 读心 -；即使低 trust，拿到明确证据也可迫使事实出现，但无法获得自愿坦白。

Milestones：

1. **Keeper:** “钟把你带回来”——不解释钟是谁造的。
2. **Witness:** 第一次死亡后承认见过“与你相似的人”。
3. **Accomplice:** 玩家带回控制台 seal 后承认参与机枢，但淡化删忆。
4. **Confession:** 玩家不强制读取其 Echo，且公开账本时保留复杂真相；Maelin 自愿交出未寄供词。
5. **Boundary:** Chorus 前，他主动把召回控制权交还玩家并接受不被原谅。

Failure：玩家强行抽取其记忆，获得证据却永久失去 `maelin_autonomy`；Crown 更容易，Chorus 关闭。
Survival：Act IV 守护钟锚，且未让 Curator 利用其 guilt 令其自断。
Autonomous resolution：自愿坦白、放弃单方面召回权限。
Gameplay impact：复活解释、run history 回放、memory tier UI；关系高时允许在死亡结算前选择保留一条非物品“问题线索”，不是直接保留装备。

### Ilyra Venn / 伊莱拉·温 — Cartographer of Unmade Roads / 未成路的绘图师

- **Surface role:** 地图、FOV 与 secret room 顾问。
- **Concealed history:** 故乡 Leth Arin / 莱斯·阿林因拒绝接入机枢被官方地图抹除；她每轮都更难记住家人的脸。
- **Desire:** 证明故乡存在，并画一张不会把人压扁成资源点的地图。
- **Fear:** 最后只剩“有用的导航能力”，忘记自己为何画图。
- **Contradiction:** 她反对官方删改，却会为了完成地图诱导玩家进入危险记忆。
- **Voice:** 具体方向、材质和距离；情绪强烈时反而只描述地形。
- **Motif:** 地图边缘画着没有坐标的厨房窗。

Milestones：

1. **Surveyor:** 把玩家当一次性探针。
2. **Missing Road:** 发现地图自动避开故乡房间，第一次请求玩家违背生成指引。
3. **Useful Lie:** 提议让尖塔制造一份清晰但虚假的家乡记忆，以换永久地图 buff。
4. **Last Map:** 玩家拒绝伪造并接受地图永久留白；Ilyra 自己选择记下“我不再记得，但我知道曾经爱过”。
5. **Navigator by Consent:** Act IV 她只标出愿意承担的路径，允许玩家拒绝最短路线。

Failure：接受 `ilyra_memory_forged`，她存活且功能更强，却失去自主身份；Crown 获得完美地图。
Survival：完成故乡 secret beat，并在 Act IV 不把她连接为自动 path oracle。
Autonomous resolution：允许不确定性存在，由她决定最后一张地图公开什么。
Gameplay impact：探索记忆、secret hint、camera look-ahead；自主路线提供“标记风险来源”而非直接揭全图。

### Vey Ashhand / 灰手维 — The Smith / 铸师

- **Surface role:** 锻造、词缀重铸与 Echo Essence 使用入口。
- **Concealed history:** 由一队在 Leth Arin 镇压中同时拒绝前进的士兵最后念头凝成；王室档案没有把他们当人。
- **Desire:** 创造不以战争用途定义的东西，并决定哪些 Echo 可以被加工。
- **Fear:** 玩家只在需要数值时来找“商店”。
- **Contradiction:** 厌恶工具化，却以工作回避讨论构成自己的多重声音。
- **Voice:** 短、触觉化，评价金属温度而非角色道德。
- **Motif:** 五指不一致的手套，每指来自不同制服。

Milestones：

1. **Service:** 只报成本，不给自己的名字。
2. **Residue:** 拒绝熔化带明确人格的 Echo；玩家可尊重或强迫。
3. **Many Hands:** 承认自己是多人形成的新“一个人”，不接受被拆回材料。
4. **First Useless Thing:** 玩家提供资源制作一只不带属性的风铃，解锁其完整对话而非战力。
5. **Covenant Forge:** Vey 自愿设计可撤回 consent token，并保留停止工作的权利。

Failure：强迫 Vey 熔化人格 Echo，获得强力 unique affix，设置 `vey_weaponized`。
Survival：不把 Vey 作为 Curator link 的永久燃料，并在熔炉失控 beat 中让其自行选择撤离。
Autonomous resolution：承认其单一人格与多重来源，完成无用之物。
Gameplay impact：锻造、词缀、Essence ethical classification；自主路线解锁透明重铸（预览代价、可撤销一次），而非无条件折扣。

### Sister Oryn / 奥林修女 — The False Archivist / 失实档案官

- **Surface role:** Codex 编目与 lore completion 解释。
- **Concealed history:** 编写王室“自愿净化”语言、删除受害者姓名；灾难后把自己写成反抗者。
- **Desire:** 留下一份终于不会服务胜者的档案。
- **Fear:** 一旦署真实姓名，她的证词只会被当作自我开脱。
- **Contradiction:** 最擅长识别宣传，也最会为自己寻找语法上的无罪。
- **Voice:** 初期被动句和机构名词；坦白后恢复第一人称与具体动词。
- **Motif:** 每次改注都留下旧墨，不把错误擦净。

Milestones：

1. **Editor:** 为玩家解释官方条目，不说明自己作者身份。
2. **Variant Hand:** 玩家比对两个版本发现相同笔迹。
3. **Grammar of Innocence:** 她承认“文本经过我”，仍说命令来自上级。
4. **Signed Testimony:** 玩家公开完整账本而非只挑有利证据后，她以真名签署删改清单。
5. **Living Archive:** Chorus 路线中，她建立允许反驳、保留来源和撤回隐私的 Codex 规则。

Failure：玩家要求她伪造一份能更快开启结局的“完整证明”，得到 lore ratio bonus 但失去证据可信链。
Survival：Act IV 保护原始与修订档案两套索引，不让 Curator 用其羞耻诱导自我删除。
Autonomous resolution：自愿署名、接受审判而不要求预先宽恕。
Gameplay impact：Codex source comparison、contradiction link 和 ending evidence explanation。

### Moth / 飞蛾 — The Unclaimed Child / 无归属之子

- **Surface role:** 以借来的短句提示 Void 与 secret；最初像一个不会完整说话的孩子。
- **Concealed history:** 不是某个死去儿童，而是多段被删除记忆在相互照料中产生的新人格。孩童外形来自“需要被保护”的共同图像，不定义年龄或能力。
- **Desire:** 被承认为现在的人，而不是旧记忆所有者的容器、Void 代表或开门钥匙。
- **Fear:** 无论王室、Aster 还是玩家，都会因需要一个答案而替 Moth 发言。
- **Contradiction:** 要求别人尊重其沉默，却早期会借用他人句子而未能请求同意；其成长也包含学习边界。
- **Voice:** 初期引用带 source cadence；中期指出“这句话不是我的”；后期语言简单但完全自选。
- **Motif:** 不扑火的黑蛾，停在没有文字的标签上。

Milestones：

1. **Borrowed Words:** 只能复述玩家已发现 lore 的片段。
2. **No:** 第一次完整自有词是拒绝玩家读取。
3. **Not Their Sum:** 玩家不以奖励为由强迫后，Moth 解释新人格的出现。
4. **Chosen Silence:** Moth 可选择一轮不提供提示；尊重它是 autonomy condition。
5. **One Voice Among Many:** Act IV 明确拒绝代表全部 Void，只为自己的 consent link 作证。

Failure：将 Moth 接入 Spire 当 universal key；它可能肉体/形态存续，却失去独立 voice，通向 Crown 变体。
Survival：不在 Final fight 用 Moth 吸收超容量记忆，且保留其断开 link 的能力。
Autonomous resolution：玩家尊重至少两次明确拒绝，Moth 选择自己的称呼与发言范围。
Gameplay impact：secret narrative resonance、来源提示、Void language translation；自主路线只在 Moth 选择时提示，不成为常驻透视外挂。

## Act I Story Boss: Caedmon Rook

Caedmon 曾是王室第七门卫长。女儿 Lysa Rook / 莱莎·鲁克死于一次本可避免的宫廷疏散；Caedmon 发现命令优先保护机枢设备，准备公开证据。王室以“创伤使其不适任”为由抽走悲痛和女儿名字，只留下服从与无对象的罪疚。Aster 知道此事，却说服 Caedmon 暂时接受处理，以便继续守住释放计划入口。

Shattering 当夜，Caedmon 看见 Aster 进入钟楼。Aster 答应事后归还记忆，但释放失败；Caedmon 的 Phantom 被 Protocol 固定为永恒守门者。他攻击玩家，不是因为认定玩家邪恶，而是命令、背叛与缺失的爱都指向同一张脸。

### Combat phases

1. **Protocol of the Seventh Gate (100–61%):** 盾击改变站位，钟波沿直线传播，Oath Chain 惩罚连续使用同一 action。Caedmon 称玩家 Warden。
2. **The Name Under the Bell (60% interrupt):** 时间线冻结，触发 Vision。Caedmon 看见 Aster 手势并叫出名字。若玩家持有军令与纸鸟，出现条件 choice “Say her name / 说出她的名字”；否则只能质问、沉默或命令。
3. **Grief Returned or Denied (60–26%):** arena 生成缺名墓碑。说出 Lysa 会让攻击从纯敌对变为保护纸鸟 Echo，战术目标改变。
4. **Resolution window (25%):** 根据证据开放三个 resolution command；若玩家继续攻击则进入 extraction 默认路线。

### Resolutions

- **Extract:** 击碎 Phantom 并取走高纯 Echo，获得强力 Act I relic；设置 `caedmon_extracted`、Maelin trust 大降，Lysa 证词链断裂。
- **Suspended mercy:** 不杀，但保留其守门协议。短期减少 Act II elite，Caedmon 仍无知地受困；这是“看似善良但未归还自主权”的路线。
- **Restored choice:** 归还痛苦、名字与命令证据，然后后退一个行动回合，让 Caedmon 自己选择。其选择是停止守门并保留悲痛，不要求玩家杀死或宽恕他。设置 `caedmon_chose_after_restoration`，是 Chorus 必需 anti-domination flag。

Caedmon 不会因玩家说一句正确台词立刻原谅 Aster。后续 Phantom 可能帮助玩家，也可能拒绝交谈；重要的是他恢复决定权。

## Death-Loop Revelation Cadence

Memory Tier 由发现组合和关系行为推进，不由裸死亡次数推进。死亡是结算与 Hub 变化的时机。

| Tier | Name | Unlock condition examples | Narrative effect |
|---|---|---|---|
| 0 | Borrowed Duty / 借来的职责 | New profile | 公开史、基础 NPC 状态 |
| 1 | Residue / 残留 | 首次归档一个 meaningful encounter 或 Lore | NPC 承认见过其他 Warden；死亡不再被说成第一次 |
| 2 | Contradiction / 矛盾 | 找到同一事件两份立场冲突且可关联证据 | 可在对话中比较来源；Act 房间出现修订版本 |
| 3 | Complicity / 共谋 | Caedmon 结局 + Act II 扩容签章/账本组合 | 公开 Aster 制度性参与；Royal Penitent background |
| 4 | Authorship / 作者身份 | 三份 identity evidence + Act III confrontation | 明确玩家为 Aster-echo 但具新主体性；Choirless background |
| 5 | Consent / 同意 | 至少三名 NPC autonomy resolution + 可撤回协议证据 | Act IV consent rooms、Chorus 路线可被看见 |
| NG+ | Aftervoice / 余声 | 完成任一 ending | Ending 后世界变体、未选路线的反证，不重置 canon 理解 |

防 grind 规则：相同失败地点连续死亡只触发一次完整叙事变体，之后使用短反应；新内容优先由新证据、不同选择、NPC 状态或到达新 depth 激活。Echo Essence 可提供玩法解锁，但关键 truth flag 不可直接购买。

每次 death return 的节奏：

1. 一句基于死亡原因的感官回声；
2. GameManager 完成 Essence/档案事务；
3. Maelin 的召回反应随 memory tier 变化；
4. 最多两名相关 NPC 主动变化，避免 Hub 每次变成台词清单；
5. 新 unlock 以角色行为呈现，再在 UI 说明效果；
6. 玩家可以立即再次出发，不被强制听完重复长对话。

## Narrative Delivery Channels

### Environmental storytelling

环境先展示结果，再提供解释。一个被锁在病床旁的召回铃，比“王国后来滥用技术”的碑文更有效。Interactable 应有物质状态、使用痕迹和位置逻辑；随机摆放的血迹日记不能替代场景构图。

### Lore Fragments

每条 Lore 有具体作者/来源、写作目的和缺口。正文长度建议 80–220 中文字或 120–320 英文词；超过则拆成有独立发现价值的条目。拾取时显示 1–3 行关键 fragment，完整版本进入 Codex。取得后至少影响一个 recontextualization、condition、关系或玩法提示。

### Memory Echoes and Visions

Vision 时长目标 8–25 秒，可被跳过且跳过后仍提交事实。使用有限色 pixel tableaux、关键声音与一到三个可控步伐/观察点，避免长电影。Vision 不替玩家解释感受；结束后的选择可以是“你相信什么/你愿意怎么用”，不是“你看见了什么”的复述题。

### Phantom Echo dialogue

Phantom 有记忆边界，不能回答死后事件。重复遭遇可显示 reconstruction drift；变化必须由尖塔/玩家状态解释。对话 choice 应改变关系、信息公开方式、Boss tactic 或 flag，不放无意义“继续询问所有选项”菜单。

### Message log

Combat channel 精确简短：`Caedmon's bell wave deals 7 resonance damage.` 叙事 channel 可以诗意但仍具体：`Ash settles inside the name someone chiseled away.` 两类有颜色/图标与筛选，不把伤害数藏在修辞中。

### Codex

Codex 同时显示来源、发现位置、互相支持/冲突条目和 Oryn 注释版本。玩家可以看见“官方版—修订版—签名证词”的演变。Completion ratio 只统计 canonical lore set，程序生成风味文本不稀释结局比例。

### StoryBeat Rooms

每个房间包含一个可在无文本时读出的视觉命题、一个可互动证据、一个可能的角色/战术后果。Required beat 推进主线；optional beat 深化人物、关系或世界。房间不能只是“普通房 + 发光纸张”。

## Choice and Relationship Design

Choice label 描述行动，不贴 `[善良]`、`[邪恶]`。好的选择让玩家在两个真实价值间决定：立即救人但延续控制、公开完整证据但伤害仍活着的人、保留痛苦但恢复自主权。

每个重大 choice 数据应注明：

- 玩家当下可知的信息；
- immediate mechanical consequence；
- affected actors 与 relationship delta；
- typed flags/decision tags；
- delayed room/dialogue/ending consequences；
- 是否可撤回、由谁撤回；
- 选择未出现时缺少的 condition explanation。

Relationship 不是好感礼物槽。Maelin 重视证据与克制，Ilyra 重视不伪造，Vey 重视用途之外的承认，Oryn 重视完整来源，Moth 重视拒绝权。同一选择可能让一人信任上升、另一人下降。达到高分不能覆盖明确的 autonomy violation。

三次关键 anti-domination opportunities：

1. **Caedmon:** 不抽取、不维持其有用失忆，归还后让他选择。
2. **Ilyra:** 拒绝制造清晰但虚假的故乡记忆，接受地图留白。
3. **Aster-in-Glass:** 不用覆盖消灭当前/旧人格，建立有限独立边界。

它们分别牺牲强力 relic、完美地图和完整 Warden 权限，确保 True Ending 条件具有可感代价。

NPC 生死应尽量来自可读风险与选择，不用不可预见即死陷阱。若 NPC 已死亡，游戏仍提供相应主线信息的替代来源，但失去其独特关系、系统能力和 Chorus link；玩家不会因单一随机房间永久卡主线。

## Ending Conditions

Ending 条件由 EndingResource 数据评估，UI 显示叙事解释而非隐藏数值猜谜。四个结局都必须有完整尾声，不把非 True Ending 写成“坏结局，请重玩”。

| Ending | Core qualification | Ethical action | Principal cost | Variants |
|---|---|---|---|---|
| Mend / 缝合 | 到达核心；接受 anchorship；锚结构可用 | 以自身持续封住裂口 | 当前人格与姓名从他人记忆中逐渐消失 | NPC trust 高时留下无名影响；self-erasure tag 高时更冷峻 |
| Sever / 割断 | 选择 severance route；掌握中央托管的压迫证据 | 一次切断 Veil，让记忆回到现实 | 未准备者承受真相洪流，尖塔庇护消失 | 完整账本与幸存 NPC 可降低混乱；鲁莽变体更惨烈 |
| Crown / 加冕 | 接受 Spire root privilege；击败/合并 Curator | 玩家集中管理保存与归还 | 善意逐渐变成优化与剥夺退出权 | NG+ Sanctum 按玩家决策被“温柔改良” |
| Chorus / 合唱 | 见下方全部条件 | 建立可拒绝、可撤回、分布式记忆契约 | 没有完美稳定；所有参与者长期承担协商与照护 | NPC 个别 epilogue 按关系与选择变化 |

### Chorus exact initial target

必须同时满足：

- `canonical_lore_ratio >= 0.85`；
- 五个 survival flags 全部为 true：Maelin、Ilyra、Vey、Oryn、Moth；
- 五个 autonomy flags 全部为 true；
- 三个 anti-domination flags 全部为 true：`caedmon_chose_after_restoration`、`ilyra_memory_not_forged`、`aster_echo_not_overwritten`；
- 三份 identity evidence：Aster 的 designer hand、release intent、Spire reconstruction proof；
- 公开 Act II 完整账本，既保留受益者也保留受害者证词；
- Moth 在 Final Assembly 明确选择加入，不能由玩家代为提交；
- 未设置不可逆 `vey_weaponized`、`oryn_forged_proof` 或 `moth_used_as_key`。

85% 是初始 authored threshold，未来由 EndingResource/Balance 数据调整，不硬编码在 NarrativeSystem。剩余 15% 允许玩家遗漏非关键区域，True Ending 不要求清空每个随机角落。

### Epilogue principles

结局展示具体生活：谁记得厨房窗、谁仍能拒绝锻造、谁给档案署名。不要用宇宙旁白宣布哲学答案。Mend 可以壮丽又令人痛；Sever 可以自由又危险；Crown 可以高效、诱人并逐渐可怕；Chorus 可以充满希望但承认维护成本。

## Bilingual Writing Style

### Shared principles

- 先写可触摸的事物，再写抽象意义。
- 每个角色有句法、比喻来源和回避方式。
- “记忆、回响、帷幕、碎片、虚空”是术语，不可每句堆叠。
- 危险中短句；安全 Hub 可留停顿，但不让所有人说同一种诗。
- 选择文本使用动词并提示对象：`Return Lysa's name` / `归还莱莎的名字`。

### Chinese / 简体中文

偏好具体动词、受控意象、清楚主语。避免翻译腔“关于……这一事实”、滥用破折号和每段都以哲理收尾。战斗句 12–24 字为宜；Dialogue 单页目标不超过 72 个汉字、最多 3 个可见文本行；choice 最多 4 个，每项建议不超过 24 个汉字。

弱：`黑暗的帷幕发出难以名状的低语，仿佛无数破碎记忆在诉说永恒的悲伤。`
强：`帷幕后有人念出一串姓名。念到第三个，钟把声音吞了。`

弱句只有氛围词；强句提供动作、数量和可追问的异常。

### English

Prefer lucid lyricism, concrete nouns, and contractions appropriate to the speaker. Avoid faux-archaic diction, ornamental inversion, and strings of abstract nouns. Combat lines target 45–80 characters; a dialogue page targets 160–180 Latin characters over no more than three visible lines; choices target 52 characters.

Weak: `Lo, beyond the sundered Veil doth sorrow eternal whisper unto thee.`
Strong: `Someone behind the Veil recites names. On the third, the bell swallows her voice.`

Translation preserves dramatic function rather than word order. Chinese can omit a pronoun when source remains clear; English restores it. Proper nouns follow glossary exactly. Translator notes record puns, source cadence and whether a borrowed Moth line must echo an earlier entry.

### Typewriter and accessibility

Default reveal speed targets 24 CJK glyphs/s and 42 Latin glyphs/s; punctuation adds small authored pauses. Confirm completes the page before advancing. Players can set instant text, hold-to-fast-forward, larger font layout and reduced flashing. Meaning never depends only on text color or typewriter timing.

## Continuity and Reveal Rules

| Topic | Act I | Act II | Act III | Act IV |
|---|---|---|---|---|
| Mercy Engine | 可知是医疗设施；滥用只见局部 | 明确国家网络与记忆税 | 明确 reconstruction 功能 | 明确可改造成 consent system |
| Aster identity | 手势、称呼、Caedmon 认出；不下结论 | 明确 Aster 参与扩容与释放计划 | 明确玩家是 Aster-echo、新人格成立 | 讨论承担与权限，不再靠身份悬念 |
| Maelin | 温柔回避、见过相似者 | 承认共同设计 | 完整看见删改当前玩家的决定 | 自愿放弃召回控制或走向失败 |
| Void | 官方敌人；出现具人性短句 | 显示被征收记忆在其中 | 明确多意识与 Moth 来源 | 明确不是单一意志、建立/拒绝契约 |
| Primordial Echo | 被称修补材料 | 揭示为原始证词 | 揭示占有证词也有伦理问题 | 决定归还、释放、统治或托管 |
| Current Warden | 最后守望者假设 | 对制度有不合理熟悉 | 新主体性被证明 | 用实际选择定义身份 |

Continuity checks：

- Caedmon 只有在 60% Vision 或玩家提前拥有特定跨轮证据时说出 Aster 全名。
- Oryn 的早期 Codex 不能使用她后期才承认的第一人称罪责语法。
- Moth 说出的借用句必须已由玩家发现，或标记为来自另一个明确 Void source；不能凭空剧透。
- 若 NPC 死亡，后续台词、system unlock 和 ending explanation 使用死亡变体，不让其无说明出现在 Hub。
- StoryBeat cooldown 防止同一大揭露在相邻 floor 重复；替代房间只给回顾或不同视角。
- NG+ 角色可记得 ending aftervoice，但不得把前一 ending 当当前 timeline 已发生事实；用 Echo possibility 描述。
- Lore localization 两种语言必须表达同一证据，不允许某语言额外泄露真相。

揭露自检问题：玩家能指出至少两条先前证据吗？新事实是否改变一个当前决定？来源为何现在能被访问？谁从该解释获益？若删掉最后 monologue，玩家是否仍能推导核心真相？任一答案为否，应重写铺垫。

### Act I quest

**`base.quest.act1.a_name_returned` — “A Name Returned / 归还一个名字”**

Stages：`unstarted -> found_order -> found_bird -> confronted_caedmon -> extracted|suspended|restored`。军令和纸鸟可不同顺序发现；缺一仍可完成前两种 resolution。Restored 需要两份证据和未破坏 recall bell frequency。

Rewards 不只给数值：

- Extracted：高强度 relic `Oathshard`、Maelin -25、Vey 拒绝评论一次、Crown affinity；
- Suspended：门卫通行 boon、Maelin +5、Ilyra -10、Quest 标记 unresolved harm；
- Restored：无即时 Boss relic，解锁 Caedmon testimony、Maelin -10 后可恢复、Moth 新自有句、Royal Penitent background 与 Chorus flag。

### Death-return variants

- Maelin：根据 burn、trap、Caedmon oath 或 memory bleed 使用不同短句；第二次起不再说“第一次”。
- Ilyra：在地图上标死亡位置；连续同点死亡后不是嘲讽，而是质疑路径是否在诱导。
- Vey：检查装备断口；若死于拆铃后的 resonance，会指出“你把求救声熔成了护甲”。
- Oryn：把死亡记成统计，玩家质疑后逐步改用姓名。
- Moth：初次只借用 Lore 句；尊重其沉默后，可能只把纸鸟放回玩家手边。

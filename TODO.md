# Development Roadmap

## P0 — Release maintenance

- Expand automated accessibility coverage for every localized modal.
- Add signed Android and notarized macOS artifacts when protected signing credentials are available.
- Grow migration fixtures before changing ProfileSave or RunSave schemas.

## P1 — Act II: Gilded Ossuary

- Add memory-tax vault generation, royal court enemies, and Seraphine Boss route.
- Implement Oryn's signed-testimony arc and Ilyra's erased-homeland StoryBeat chain.
- Add ranged weapons, summons, additional elite modifiers, and biome-specific traps.

## P2 — Acts III and IV

- Build Mirror Deep reconstruction variants and Aster-in-Glass confrontation.
- Build Wound Crown consent-link finale and all four playable ending sequences.
- Add New Game+ Aftervoice variants and ending-reactive Sanctum states.

## P3 — Mod ecosystem

- Publish JSON Schemas and an editor plugin for IDs, references, dialogue graphs, and localization parity.
- Add content-pack discovery, dependency diagnostics, explicit override UI, and safe pack quarantine.
- Provide a standalone validator and example community content pack.

## P4 — Long-term and DLC

- Optional fifth Act DLC exploring memories voluntarily surrendered after the Shattering.
- Additional starting backgrounds, challenge seeds, daily runs, and accessibility presets.
- More endings centered on partial regional covenants and contested archives rather than a single global solution.
